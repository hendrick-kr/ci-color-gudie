CI Color Swatch Generator 가이드

🎨 CI Color는 브랜드 색상을 쉽게 관리하고, 다양한 형식(HEX, RGB, CMYK)의 정보가 담긴 색상 견본(Swatch) 이미지를 다운로드할 수 있는 도구입니다.

👉 바로 사용하기

1. 색상 추가하기

왼쪽 패널에서 두 가지 방식으로 색상을 추가할 수 있습니다.

방법 1: HEX 코드 직접 입력

  1. HEX INPUT 탭을 선택합니다.
  2. 입력창에 #FF5733 같은 HEX 코드를 입력합니다.
  3. ADD COLOR 버튼을 클릭하면 팔레트에 색상이 추가됩니다.

방법 2: 색상 피커 사용

  1. PALETTE 탭을 선택합니다.
  2. 원형 컬러 피커에서 원하는 색조(Hue)와 채도(Saturation)를 선택합니다.
  3. 아래 BRIGHTNESS 슬라이더로 밝기를 조절합니다.
  4. ADD SELECTION 버튼을 클릭합니다.
💡 팁: RGB 또는 CMYK 슬라이더를 직접 조절하여 더 정밀하게 색상을 만들 수도 있습니다.

2. 팔레트 관리하기

추가된 색상은 오른쪽 MY PALETTE 영역에 카드로 표시됩니다.

동작 방법
선택/해제 색상 카드를 클릭
HEX 복사 카드 위에 마우스를 올리고 복사 아이콘 클릭
개별 삭제 휴지통 아이콘 클릭
전체 삭제 상단 CLEAR 버튼 클릭
✨ 자동 저장: 팔레트는 브라우저에 30일간 자동 저장되므로, 창을 닫았다가 다시 열어도 그대로 유지됩니다.

3. 이미지 내보내기 설정

이미지를 다운로드하기 전, 왼쪽 하단의 EXPORT CONFIG에서 견본 이미지의 스타일을 설정하세요.

설정 옵션

  • OUTPUT RESOLUTION: 이미지 크기
    • 256 x 256 — 웹/SNS용
    • 512 x 512 — 일반 용도
    • 1024 x 1024 — 고해상도
    • 2048 x 2048 — 인쇄용
  • TEXT COLOR: 텍스트 색상 (AUTO는 배경색에 맞게 자동 조절)
  • TEXT POSITION: 텍스트 위치 (내부 상단/중앙/하단, 외부 등)
    INSIDE (내부)
    OUTSIDE (외부)
  • INCLUDE VALUES: 이미지에 표시할 정보
    • ✅ HEX CODE — #FF5733
    • ✅ RGB VALUE — RGB: 255, 87, 51
    • ✅ CMYK VALUE — CMYK: 0, 66, 80, 0

4. 이미지 다운로드

개별 다운로드

색상 카드 위에 마우스를 올리고 다운로드 아이콘을 클릭하면 해당 색상의 PNG 이미지가 저장됩니다.

여러 색상 일괄 다운로드

  1. 다운로드할 색상 카드들을 클릭하여 선택합니다 (체크 표시가 나타남).
  2. 상단의 DOWNLOAD SELECTED 버튼을 클릭합니다.
  3. 선택한 모든 색상이 순차적으로 다운로드됩니다.

💡 활용

  • 🎨 브랜드 가이드라인 문서 제작 시 색상 견본 이미지로 활용
  • 📊 PPT/기획서에 정확한 색상 코드와 함께 삽입
  • 🖨️ 인쇄물 제작 시 CMYK 값 확인 및 전달
  • 💻 개발자 협업 시 정확한 HEX/RGB 값 공유
